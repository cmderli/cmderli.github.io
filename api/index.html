访问以上几个网站的/均会302至https://cmderli.net/api/(也就是本页面)。
如果您确信您没有点击本页面或以上三个域名的主页的链接，或者本页面的URL不是https://cmderli.net/api/，那么您可能遇到了DNS劫持。

关于

这个网站是我突发奇想为了给大家提供免费的api服务而设置。
该服务托管于Vercel。serverless对于这个服务来说已经够了，不需要额外花钱钱买VPS。
由于Vercel对免费用户的配额只有每月100GB流量，所以请您量力而行，不要一下子发那么多Request，更不要滥用。
如果您有大请求量需求的话，请您自行搭建本服务，本服务的源代码之后会发布。
以下是现公开的几个api。

Bing 每日美图

获取 Bing 的每日美图。

返回：image/jpeg (302到bing的今日美图)。

预览

IP查询

获取您的外网IP。

返回：text/plain (您的外网ip)。
该功能和test.ipw.cn不能说很像，只能说一模一样。

IP属地等相关信息查询

查询IP属地、属地经纬度、ISP、时区。

返回：text/plain (JSON)
该接口使用了ipinfo.io的接口，免费用户每月配额50k次请求，悠着点用。
您可以加上?ip=参数，以指定要查询的ip。
